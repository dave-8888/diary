# 日记 Windows 安装与使用说明

## 1. 安装包位置

- Windows 安装包输出目录：`dist/windows-installer/`
- 当前试行版安装包命名规则：`Diary-Setup-v版本号.exe`

## 2. 安装步骤

1. 双击运行安装包 `.exe`。
2. 按向导点击“下一步”。
3. 选择安装目录，默认安装到 `C:\Program Files\Diary MVP\`。
4. 如果需要桌面快捷方式，勾选“创建桌面快捷方式”。
5. 点击“安装”，等待完成。
6. 安装完成后，可直接勾选“启动 日记”并打开程序。

## 3. 首次使用

1. 打开应用后，左侧栏可进入：首页、写日记、时间轴、垃圾桶、设置。
2. 建议先打开“设置”，按需要调整主题、语言、应用名称和图标。
3. 如果要使用语音转文字，请在“设置”中填写 OpenAI API Key。

## 4. 常用操作

### 4.1 写日记

1. 点击左侧“写日记”。
2. 输入标题和正文。
3. 可使用媒体工具栏导入图片、拍照、录视频、录音、转写最新录音。
4. 可填写位置、选择心情、添加标签。
5. 页面标题“今天发生了什么？”右侧提供：
   - `导出日记`：将当前日记导出到本地文件夹。
   - `保存日记`：保存或更新当前日记。

### 4.2 首页

1. 首页展示最近日记和主题化概览区。
2. 若设置了标签，可在首页按标签筛选。
3. 点击日记卡片可进入编辑。

### 4.3 时间轴

1. 点击左侧“时间轴”查看全部日记。
2. 可结合标签筛选查看特定内容。
3. 点击任意日记卡片可继续编辑。

### 4.4 垃圾桶

1. 被删除的日记会先进入垃圾桶。
2. 可单条恢复，也可批量恢复。
3. 可清空垃圾桶，清空后将无法恢复。

### 4.5 设置

可在设置页进行以下操作：

- 切换主题
- 切换语言
- 修改应用名称
- 修改应用图标 / Windows 外层图标
- 配置语音转写 API Key
- 管理心情库
- 导入导出迁移包

## 5. 数据保存位置

程序数据默认保存在系统文档目录下：

- `diary_mvp/db/diary.db`
- `diary_mvp/user_data/diary/images/`
- `diary_mvp/user_data/diary/audio/`
- `diary_mvp/settings/`

建议定期备份以上目录。

## 6. 重新安装是否保留原有数据

- 会保留。
- 安装包只会更新程序文件，不会把示例数据或开发机数据一起带进去。
- 日记数据库、图片、录音、设置都保存在系统文档目录下，不在安装目录里。
- 如果电脑上已经安装过旧版本，直接再次安装即可，原有日记数据不会因为覆盖安装而丢失。

## 7. 重新打包发布版

仓库已提供一键打包脚本：

```powershell
powershell -ExecutionPolicy Bypass -File .\tools\build_windows_installer.ps1
```

说明：

- 脚本会先执行 `flutter build windows --release`
- 然后调用 Inno Setup 生成安装包
- 默认读取当前仓库最近的 git tag 作为安装包版本名

如果只想跳过 Flutter 构建、直接重打安装包：

```powershell
powershell -ExecutionPolicy Bypass -File .\tools\build_windows_installer.ps1 -SkipBuild
```

如果想手动指定发布标签：

```powershell
powershell -ExecutionPolicy Bypass -File .\tools\build_windows_installer.ps1 -ReleaseTag v0.1.1-beta.1
```

## 8. 注意事项

- Windows 安装包仅适用于 64 位 Windows。
- 若启用语音转文字，需要网络连接和有效的 OpenAI API Key。
- 清空垃圾桶前请先确认，操作不可恢复。
